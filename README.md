# Smart-Matix-Headlight-System
Smart Matrix Headlight System using Arduino UNO, ultrasonic sensors, and Bluetooth. Detects obstacles and auto-adjusts headlights via servo motor, with manual control through a mobile app. Enhances night-time safety and supports autonomous/RC vehicles.
